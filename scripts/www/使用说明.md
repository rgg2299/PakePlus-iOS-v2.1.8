# 讯飞输入法按键编辑器 - 使用说明

## 📖 简介

这是一个专门用于制作讯飞输入法皮肤按键背景图的在线编辑器。通过这个工具，你可以快速制作出美观的按键背景，然后导出为PNG图片，用于讯飞输入法皮肤的定制。

## ✨ 核心功能

### 1. 实时预览
- 顶部预览区实时显示按键效果
- 所有参数调整都会立即反映在预览中

### 2. 预设模板（40套）
内置40种精美的预设模板，包括：
- 霓虹紫、珊瑚橙、薄荷绿、深海蓝
- 玫瑰金、暗黑模式、玻璃效果、金属质感
- 简约白、活力黄、优雅粉、自然绿
- 科技蓝、深紫渐变、橙红渐变、青绿渐变
- 宝石红、海洋蓝、柠檬黄、薰衣草
- 日出橙、森林绿、黄昏紫、石墨黑
- 珍珠白、玫瑰粉、天空蓝、大地棕
- 冰晶蓝、火焰红、翡翠绿、琥珀黄
- 紫水晶、珍珠母、钢铁灰、樱花粉

### 3. 基本设置
- **图片尺寸**：50-300像素，可自由调整
- **圆角半径**：支持统一设置和单独设置四个角
- **边缘留白**：为图片添加边缘空白区域
- **整体透明度**：0-100%透明度调节

### 4. 填充颜色
- **纯色填充**：单一颜色填充
- **渐变填充**：
  - 线性渐变：支持0-360度角度调节
  - 径向渐变：从中心向外扩散
  - 支持透明度调节

### 5. 描边设置
- 可启用/禁用描边
- 描边宽度：0-10像素
- 描边颜色：支持透明度
- 描边类型：外描边、内描边、居中描边

### 6. 阴影设置
- 可启用/禁用阴影
- 阴影模糊：0-20像素
- 水平/垂直偏移：-20到20像素
- 阴影颜色：支持透明度
- 阴影类型：外阴影、内阴影

## 🚀 使用方法

### 步骤1：打开编辑器
直接在浏览器中打开 `index.html` 文件

### 步骤2：选择模板或自定义
- 点击预设模板快速应用样式
- 或者手动调整各项参数自定义效果

### 步骤3：调整参数
根据需要调整以下参数：
- 尺寸、圆角、留白、透明度
- 填充颜色（纯色或渐变）
- 描边设置
- 阴影设置

### 步骤4：导出图片
- 点击底部"下载图片"按钮
- 图片将自动保存为 `xunfei_btn_时间戳.png`
- 格式为PNG，支持透明背景

### 步骤5：重置设置
- 点击"重置"按钮恢复默认设置

## 📱 响应式设计
编辑器支持多种屏幕尺寸：
- 桌面端：完整显示所有控件
- 移动端：优化的触摸体验

## 📋 常见问题

### Q1：导出的图片为什么有透明背景？
A：这是正常的，PNG格式支持透明背景，方便后续叠加使用

### Q2：如何制作正圆形按键？
A：将圆角半径设置为尺寸的一半即可

### Q3：如何复制某个模板的配色？
A：点击模板后，可以在右侧看到各项参数值

### Q4：支持哪些浏览器？
A：支持所有现代浏览器（Chrome、Firefox、Edge、Safari）

### Q5：可以保存我的设置吗？
A：目前不支持云端保存，建议截图记录或使用浏览器书签

## 🎨 皮肤制作流程

制作讯飞输入法皮肤的完整流程：

1. **使用此编辑器制作按键背景图**
   - 设计普通键、功能键、删除键等不同样式
   - 导出为PNG格式

2. **创建皮肤文件夹结构**
   ```
   皮肤名称/
   ├── info.ini              (皮肤信息)
   ├── config/
   │   ├── style.ini         (按键样式)
   │   ├── anim.ini          (动画效果)
   │   └── anim-style.ini    (动画样式绑定)
   └── res/
       ├── image/            (放入导出的按键图片)
       ├── font/             (字体文件)
       └── sound/            (音效文件)
   ```

3. **配置INI文件**
   - 根据讯飞输入法皮肤规范编辑配置文件

4. **打包成IT格式**
   - 使用讯飞输入法皮肤制作工具打包
